# SODA Migration (ICX → SODA)

**Migrate self-custody ICX 1:1 to SODA.** ICON Republic Token (ICX) holders can migrate to SODA at a 1-to-1 ratio. Same community, fresh tokenomics.

## What it does

The migration interface accepts ICX from a self-custody wallet and issues SODA at a 1:1 ratio. The migration is permanent — migrated ICX is consumed and the equivalent SODA supply is issued. CEX-held ICX is migrated automatically by supporting custodians; this app is for self-custody users.

## Key facts

- **1:1 ICX → SODA ratio**.
- **Self-custody only** — for ICX held in user-controlled wallets. ICX held on supporting centralized exchanges is migrated automatically by those custodians.
- **Permanent migration** — no reverse path back to ICX.
- **CEX support** rolling out — centralized exchanges support migration for custodied balances; user action not required for those holdings.
