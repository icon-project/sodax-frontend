# SODAX Swap

**Cross-network token swaps powered by the SODAX solver and intent system.** Swap between any supported asset on any supported network in a single user action — no bridges to manage, no manual routing.

## What it does

The user expresses an intent ("swap X of asset A on chain 1 for asset B on chain 2"). A solver fulfills the intent atomically. Settlement happens on both source and destination networks. Users never interact with bridges directly.

## Key facts

- **Cross-network swaps** between any pair of supported networks.
- **18+ networks** supported (Ethereum, Base, BNB Chain, Polygon, Optimism, Arbitrum, Avalanche, Kaia, ICON, Injective, Nibiru, Solana, Sonic, Stellar, Sui, HyperEVM, Bitcoin, Near, Stacks, Redbelly, LightLink).
- **Solver-routed** for optimal pricing and depth.
- **Sub-30-second** typical settlement for most pairs.
- **Every swap burns SODA** — partner protocols collect fees that buy back and burn SODA from the market.
