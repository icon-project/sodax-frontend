# SODAX Save

**Earn yield by supplying assets to the SODAX money market.** Supply supported tokens from any of the 18+ networks; earn yield based on real-time supply and borrow demand.

## What it does

Save lets users deposit supported assets into the SODAX money market. Suppliers earn interest paid by borrowers. Yields are variable, set by utilization of each asset.

## Key facts

- **Cross-network supply** — deposit assets from any supported network without bridging first.
- **Money market reserve assets** — borrowing demand on the same assets generates supplier yield.
- **18+ supported networks**.
- **Withdrawals** — supplied assets can be withdrawn at any time, subject to available liquidity in the reserve.
