# SODAX Borrow

**Borrow against supplied collateral on the SODAX money market.** Use assets supplied via SODA Save as collateral to borrow other supported assets across networks.

## What it does

Borrow draws against the user's collateral position in the SODAX money market. Variable-rate borrowing, with interest paid back to suppliers. Cross-network: collateral on one chain can back borrows of an asset on another.

## Key facts

- **Cross-network borrowing** — borrow assets on any supported network using collateral from any other supported network.
- **Variable interest rates** set by utilization on each asset.
- **Liquidation** if collateral value falls below required ratios.
- **18+ supported networks**.
