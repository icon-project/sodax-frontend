# SODAX SDK

**Infrastructure for cross-network DeFi.** Give your users access to over 50 assets without setting up from scratch. SODAX SDK provides cross-network execution, money market integration, and bnUSD minting access across 18+ networks. All layers expose SODAX-coordinated execution and protocol-supported liquidity.

## Three-layer architecture

1. **Core SDK — Foundation layer** (`@sodax/sdk`) — core logic for swaps, lending, bridging, staking, migration, and partner modules. Use directly when you want maximum control.
2. **Wallet SDK — Connection layer** (`@sodax/wallet-sdk-react`) — opinionated React wrapper for cross-chain wallet providers (EVM, Solana, Sui, Stellar, Injective, ICON). Built on Zustand state and EIP-6963.
3. **dAppKit — Experience layer** (`@sodax/dapp-kit`) — pre-built React hooks combining the SDK + wallet SDK + React Query. Fastest path from zero to a working cross-network app.

## What it unlocks for partners

- **Intent-based cross-network swaps** routed by the SODAX solver.
- **Money market integration** (supply, borrow, withdraw) across all supported networks.
- **bnUSD minting** access.
- **18+ networks** through a single integration surface — Ethereum, Base, BNB Chain, Polygon, Optimism, Arbitrum, Avalanche, Kaia, ICON, Injective, Nibiru, Solana, Sonic, Stellar, Sui, HyperEVM, Bitcoin, Near, Stacks, Redbelly, LightLink.
- **Agentic hooks** — public Builders MCP server for AI-agent-powered integrations.

## Who's building with it

Partners using the SODAX SDK include Houdini, Balanced, Amped Finance, Hana Wallet, Sommelier, Superposition, RadFi, and 1inch.

## SODA tokenomics for SDK partners

SODA is a governance token with yield from staking and protocol fees. **SDK partners configure fees in any token — no SODA required for integration.**
