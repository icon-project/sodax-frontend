# SODAX

**Infrastructure for modern money.** SODAX is a cross-network execution and liquidity system that lets applications offer complex DeFi actions — swaps, lending, borrowing, staking, bridging — across any supported blockchain network as a single financial system, without becoming a cross-network infrastructure team themselves.

One SDK. Scaling DeFi products across networks.

## What SODAX provides

- **Cross-network development** — tailored to your product. Choose your integration: wallet, DEX, lending protocol, perp DEX, yield app, new network, or solver marketplace.
- **Scalable execution stack** — the system handles routing and liquidity. SODAX coordinates timing, failure recovery, and async execution across heterogeneous networks.
- **Production-ready in days** — a single SDK with agentic hooks. Public Builders MCP server exposes on-chain protocol data and SDK documentation to AI agents.
- **Live on 18+ networks** — Ethereum, Base, BNB Chain, Polygon, Optimism, Arbitrum, Avalanche, Kaia, ICON, Injective, Nibiru, Solana, Sonic, Stellar, Sui, HyperEVM, Bitcoin, Near, Stacks, Redbelly, LightLink.

## SODA token

The native utility and governance token. Supply capped at 1.5B, no emissions, deflationary by design — every swap on partner protocols across 18 networks burns SODA. 20% of protocol revenue flows to stakers.
